# 🎗 Breast Cancer Classifier

## 📌 Overview
This project classify breast tumors as malignant or benign using both supervised and unsupervised learning techniques.

## ⚙️ Process
- **Data Preprocessing:** Cleaned the dataset by handling missing values and noise to ensure data quality.
- **Feature Extraction:** Selected and transformed relevant tumor features for effective model training.
- **Exploratory Data Analysis (EDA):** Analyzed tumor characteristics and feature correlations using statistical and visual techniques.
- **Model Training:** Trained multiple models including Logistic Regression, K-Nearest Neighbors, Naive Bayes, Decision Tree, Neural Network, and K-Means Clustering.
- **Model Evaluation:** Compared models using performance metrics to assess classification accuracy.
- **Best Model Selection:** Logistic Regression achieved the highest performance with 97% accuracy, making it the final model for tumor classification.

## 🌐 Dataset Source
Dataset used in this project is **load_breast_cancer dataset**, which is an in-built dataset available in scikit-learn library commonly used for Machine Learning tasks.
