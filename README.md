# 🎗 Breast Cancer Classifier

## 📌 Overview
This project classify breast tumors as malignant or benign using Supervised Learning techniques.

## ⚙️ Process
- **Data Preprocessing:** Cleaned data by removing outliers and by checking any NULL values but dataset doesn't contain any NULL values.
- **Feature Extraction:** Selected and transformed relevant tumor features for effective model training.
- **Exploratory Data Analysis (EDA):** Analyzed tumor characteristics and feature correlations using statistical and visual techniques.
- **Model Training:** Trained multiple models including Logistic Regression, K-Nearest Neighbors, Naive Bayes, Decision Tree, & Neural Network.
- **Model Evaluation:** Compared models using performance metrics to assess classification accuracy.
- **Best Models:** Logistic Regression & Neural Network achieved the highest performance with 97% accuracy.
- **Selected Model:** Logistic Regression because it matches the neural network’s performance while being far simpler, faster, more interpretable for clinicians, and cheaper to run and maintain in production.

## 🌐 Dataset Source
Dataset used in this project is **Wisconsin Diagnostic Breast Cancer dataset**, which is an in-built dataset available in scikit-learn library commonly used for Machine Learning tasks.
